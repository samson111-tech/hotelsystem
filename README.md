Project Overview:
A desktop Hotel Management System built with Python (Tkinter) and MySQL that provides database password validation, user authentication (login), customer records, room inventory, and bookings. The GUI is implemented with Tkinter and the backend persists data in a MySQL database.
Key features include:
•	Database validation by submitting the right password to connect with MySQL
•	Login system with username/password and role support (admin/user{staff}).
•	Customer management: add, edit, delete, search customer records.
•	Room management: room types, availability, rates (prices).
